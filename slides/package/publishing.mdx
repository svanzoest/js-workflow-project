## CI tips

A commit message with `[ci skip]` will be skipped.

Ensure git version of dependencies get used
```bash
yarn install --frozen-lockfile
```

---
## Bump version

```bash
yarn config set version-git-message "$COMMIT_PREFIX %s"
yarn version --patch [--minor] [--major]
git push --follow-tags origin HEAD:$BRANCH
```

---
## NPM Publish Package

Use `@<organization>/<package>` for scope

```bash
yarn publish --tag dev
```
