## Linting

Tools to help us focus on writing code and not worry about style.

- prettier.io
  - `pretty-quick` with `husky`
- eslint
- .editorconfig

---

### .editorconfig

```
# top-most EditorConfig file
root = true

# Unix-style newlines with a newline ending every file
[*]
end_of_line = lf
insert_final_newline = true
indent_style = space
indent_size = 2
charset = utf-8
```
