## Linting

Tools to help you focus on writing code and nearly forget about style.

- [.editorconfig](https://editorconfig.org/) 
- [prettier](https://prettier.io/)
- [eslint](https://eslint.org/)
- [stylelint](https://stylelint.io/)

They all work together

---
### .editorconfig

```ini
# top-most EditorConfig file
root = true

# Unix-style newlines with a newline ending every file
[*]
end_of_line = lf
insert_final_newline = true
indent_style = space
indent_size = 2
charset = utf-8
```

---

### prettier

Focused on Code Formatting

Shared Config

- on [github](https://github.com/SwitchCaseGroup/switchcase-prettier-config)
- on [npm](https://www.npmjs.com/package/@switchcase/prettier-config)

package.json

```json
"prettier": "@switchcase/prettier-config",
```
